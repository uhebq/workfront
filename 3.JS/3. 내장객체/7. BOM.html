BOM(Browser Object Model)

1) location 객체

브라우저 주소 표시줄과 관련된 객체

location.reload()

페이지를 새로 고침하는 메소드

location.href

현재 페이지의 URL 주소를 가지고 있는 속성으로
해당 속성의 값을 변경해서 페이지를 이동할 수 있다.

location.assign() : 페이지 이동

location.replace() : 페이지 이동
history에 기록되지 않는다

2) history 객체

브라우저의 방문 기록과 관련된 객체이다. 방문한 사이트 주소가 배열 형태로 저장됨

3) navigator 객체

웹 페이지를 실행하고 있는 브라우저에 대한 정보를 가지고 있는 객체이다.

4) screen 객체

운영체제의 화면(모니터)의 정보를 가지고 있는 객체이다.
